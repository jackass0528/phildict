Proofreading instructions

Orthography The orthography of this dictionary departs in a few important points from what is normally encountered in Cebuano writing. We strictly follow the way the author has written the words. No need to indicate deviations from the common spelling (unless there is an apparent error) Accented letters Outside the Latin-1 code set, this text uses a number of special characters. A wedge (hajek) on a, i, and u [va], [vi], [vu] A macron and grave accent on a, i, and u: [=á] [=í] [=ú] Furthermore, in the introduction the symbol for glottal stop ^? and eng [ng] Arrows: Left and right pointing arrows are used, type as (<-) and (->) resp. Asterisk: Asterisks are used with a few words. Just type as such. Proofer remarks should always have two asterisks. Dagger: Type the dagger symbol as [+] Subscript numbers _1 _2 _3 _4, etc. End-of-line hyphenation: In Cebuano words, you can remove the end-of-line hyphens except when a consonant is followed by a vowel. Examples: ní-yas -> níyas; kará-ang -> karáang; BUT tan-aw -> tan-aw.
